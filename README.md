# genid

## Install

**Linux/macOS**
```sh
curl -fsSL https://raw.githubusercontent.com/PaxtonTerryDev/genid/master/install.sh | sh
```

**Windows (PowerShell)**
```powershell
irm https://raw.githubusercontent.com/PaxtonTerryDev/genid/master/install.ps1 | iex
```

Or download a binary directly from the [releases page](https://github.com/PaxtonTerryDev/genid/releases).
